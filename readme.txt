== Description ==

Integrate Wordpress and vBulletin.
This plugin allows for new Wordpress posts to be automatically posted to a specific vBulletin forum of your choosing.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set the path to your vBulletin installation in the settings page.
4. Activate the features you want.

= Will it work with a formatted post? i.e. post images from the Wordpress post to the forum =

Yes. If the forum is set to accept HTML, it will work as intended.
